# FinSight Enterprise AI

An end-to-end production AI system for financial document intelligence.

## Stack
- **Orchestration:** LangGraph, MCP tool servers
- **Automation:** n8n
- **Storage:** Supabase (PostgreSQL), Qdrant (vector DB)
- **Pipeline:** PySpark, Medallion architecture (Bronze → Silver → Gold)
- **Extraction:** Tesseract OCR, LLM extraction pipeline (Groq / Llama 3.3 70B)
- **Observability:** LangSmith, Supabase dashboard
- **Frontend:** Streamlit
